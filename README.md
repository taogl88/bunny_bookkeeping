# bunny记账

基于 Flutter 的个人记账应用，支持 Android。

**目录下有 apk 文件自行下载体验**

## 功能特性

### 基础记账
- 收入/支出记账（增删改查）
- 按日期归档，自动分组显示
- 账单搜索（金额、备注、类别、时间）

### 数据分析
- 周/月/年折线图（收支趋势）
- 月度/年度预算管理与提醒
- 记账成就面板

### 资产管理
- 资产账户管理
- 资产快照（按月记录余额）

### 自动化
- 银行短信自动记账（自动解析短信内容生成账单）

### 数据安全
- 数据导入（JSON/CSV，兼容主流记账应用格式）
- 数据导出（JSON/CSV，和导入格式一致）
- 云端自动备份（腾讯云 COS，每天/每周/每月定时）

## 技术栈

- Flutter 3.x + Dart
- Riverpod（状态管理）
- SQLite（本地数据库）
- 腾讯云 COS（云端备份存储）

## 项目结构

```
lib/
├── data/           # 静态数据（图标、分类）
├── db/             # 数据库操作
├── models/         # 数据模型
├── pages/          # 页面
├── providers/      # Riverpod 状态管理
├── services/      # 业务服务（导入/导出/备份）
├── theme/          # 主题配置
└── widgets/        # 通用组件
```

## 开发

```bash
# 安装依赖
flutter pub get

# 运行 debug 版本
flutter run

# 构建 release apk
flutter build apk --release
```
